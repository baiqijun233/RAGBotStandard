# 初始需求记录

## 用户要求

用户要求读取前一个 ChatGPT 分享对话中的项目上下文，继续整理项目，并明确提出：

- 把项目放入新的项目行列。
- 每执行一个项目，必须创建 `Project编号_项目名称` 文件夹。
- 所有相关内容必须放入对应项目文件夹，不能放到电脑其他位置。
- 命名规范、文档使用规则、文档目录结构要汇总到全局工作台。

## 项目上下文

该项目来自 PDF RAG 学习过程：

- Streamlit UI
- PDF 上传
- 文本切分
- HuggingFace embedding
- Chroma 向量库
- PDF 内容检索

当前版本是“PDF 检索器”，下一阶段可以升级为“LLM 生成回答版 RAG”。
